Demonstrasi Algoritma Sort & Search 📊

Sorting Algorithms

Bubble Sort

Membandingkan elemen yang berdekatan secara berulang

⏱️ Time: O(n²) | Space: O(1)

Selection Sort

Memilih elemen terkecil dan ditempatkan di depan

⏱️ Time: O(n²) | Space: O(1)

Insertion Sort

Menyisipkan elemen ke posisi yang tepat

⏱️ Time: O(n²) | Space: O(1)

Merge Sort

Membagi array dan menggabungkan dengan terurut

⏱️ Time: O(n log n) | Space: O(n)

Quick Sort

Membagi berdasarkan pivot dan rekursi

⏱️ Time: O(n log n) avg | Space: O(log n)

Searching Algorithms

Linear Search

Mencari elemen dengan pemeriksaan satu per satu

⏱️ Time: O(n) | Space: O(1)

Binary Search

Mencari di array terurut dengan membagi dua

⏱️ Time: O(log n) | Space: O(1)

Interpolation Search

Memperkirakan posisi elemen berdasarkan nilai

⏱️ Time: O(log log n) avg | Space: O(1)

Jump Search

Melompat dengan interval tetap lalu linear search

⏱️ Time: O(√n) | Space: O(1)

Exponential Search

Mencari range eksponensial lalu binary search

⏱️ Time: O(log n) | Space: O(1)
